# 테스트 아티클

이것은 `act`를 이용한 GitHub Actions 테스트용 아티클입니다.

## 목적

- GitHub Actions 워크플로우를 로컬에서 테스트
- 마크다운 검증 스크립트 테스트
- CI/CD 파이프라인 검증

## 사용 기술

- GitHub Actions
- act (로컬 테스트 도구)
- Node.js
- 마크다운 검증 스크립트
